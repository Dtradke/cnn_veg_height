# cnn_veg_height
A convolutional neural network to determine vegetation height from 2D aerial imagery.

Main Libraries:
Keras = 2.2.4
TensorFlow = 1.13.1
NumPy = 1.16.2
SciPy = 1.0.0
OpenCV = 3.3.0

Must build directories:
./data/
./data/_untrained/
./models/
./output/
./output/datasets/
./output/modelViz/
./output/predictions/
./output/results/
